# folk-api
Unofficial API that based on spider.
用爬的姿势将一些网站搞成api。

## api list
- baidu
    - BaiduSearch // 百度搜索
- sogou
    - SogouDict // 搜狗输入法的词典
- douban
    - DoubanBook // 豆瓣图书
        - 惊讶地发现有些图书在没有登录态的时候返回“页面不存在”。嗯。
    - DoubanMovie // 豆瓣电影
    
    
## 其他
- 爬虫的地方，对IP有反爬虫的，用了[IPProxyPool](https://github.com/qiyeboy/IPProxyPool)
